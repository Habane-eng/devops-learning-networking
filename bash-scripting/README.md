# Bash Scripting Module

## Scripts
- calculator.sh – Basic arithmetic calculator
- file_ops.sh – Directory and file automation
- file_checker.sh – File permission checker
- backup_txt.sh – Backup script with timestamps

## Key Learnings
- Bash scripting fundamentals
- File and directory automation
- Error handling and permissions
- Automation mindset for DevOps

## Why Bash Matters in DevOps
Bash is essential for automation, CI/CD pipelines, server management, and system administration.
